Stylus Record Manager

Languages: HTML, CSS, JavaScript

Technologies used: Node.js, Docker, Express, PostgreSQL, Discogs API

Vinyl collectors such as myself are always on the lookout for simple collection management solutions to help us find out more about the music we've purchased. We often like to know what country a release comes from, how rare it is and how much it is worth. Stylus utilises the Discogs API to enable a user to search for and catalogue the items they own. The user's total collection value is then calculated. Some users may also wish to use Stylus to export their collection to a PDF or CSV.

HabiPad Habit Tracker

Languages: HTML, CSS, JavaScript

Technologies used: Node.js, Docker, Express, Heroku, PostgreSQL

In recent years, there has been a renewed focus from the public on living a healthy lifestyle. This often entails forming new habits such as sleeping 8 hours or drinking 2 litres of water each day. HabiPad was a group project aimed at providing a user with a platform to define and monitor a habit of their choosing. To track a habit, the user must first create an account or login, enter their habit name and select a frequency (daily, weekly, monthly). Following successful habit creation, the user can then make frequent log entries towards their defined habit, dependent on their chosen frequency. A streak is displayed next to the habit, encouraging the user to keep making log entries.

StreetHub Community Journaling

Languages: HTML, CSS, JavaScript

Technologies used: Node.js, Express, Giphy API

Many communities use online community journalling platforms to promote events happening in their area. These platforms may also be used to raise awareness of crimes or to advertise businesses. StreetHub was a group project and provides an anonymous posting solution for members to share articles containing content of their choosing. Other users can then leave a comment on these posts, along with an optional GIF or emoji reaction.

PetPlus Pet Care

Languages: HTML, CSS, PHP, JavaScript

Technologies used: MAMP, AWS, Alexa Developer Console

When going on holiday, pet owners are often faced with the difficult task of finding someone to care for their pet if the country they are travelling to does not allow animals from overseas to enter. The individual tasked with caring for the owner's pet may need to seek veterinary advice which would usually require calling or emailing a vet and waiting for a response. PetPlus is a platform for vets to upload pet information which can be accessed by the carer through an Alexa skill. This information includes follow-up actions from previous treatments, diet plans and exercise routines. If a pet has been prescribed medication, dosage information can also be retrieved. PetPlus was developed for my university dissertation in conjunction with animal health industry professionals.
